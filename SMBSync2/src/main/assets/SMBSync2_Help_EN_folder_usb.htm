“List Directory”

Show directory list for external SDCARD.

(Selectable directory will be displayed when you press the listing button, tap the directory name. Please tap the icon to the left of the directory name to open a sub-directory.)

 

"Add keywords"

%YEAR%,%MONTH%,%DAY% and %DAY-OF-YEAR% can be used as target directory names. The keywords are converted into the year, month, and day at the time of start sync.

 

"Directory"

Specify the directory name of the master or target. If the target directory does not exist, it will be created during sync.

 

“Select USB media”

Select the USB media root directory to get access to USB media. If you can not select it please send the system information to the developer with "Menu -> System Information".

USB media can not be used unless there is a mount point (/storage/0123-0123, etc.) corresponding to the USB media UUID.
