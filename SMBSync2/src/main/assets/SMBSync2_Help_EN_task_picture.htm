When the shooting date and time can not be acquired from Exif, a confirmation message is displayed

If checked, when the shooting date and time can not be acquired from Exif, a confirmation message is displayed to the user as to whether or not to use the last update time of the file. If you select Cancel as a confirmation message, the file will not be archived.

Choose the criteria for determining by shooting date the photos you want to archive.

       Immediate all

To archive all pictures

       Older than 7 days

The shooting date is older than the current time by 7 days or more

       Older than 30 days

The shooting date is older than the current time by 30 days or more

       Older than 60 days

       Older than 90 days

       Older than 1 years

        Change the file name when archiving

The file name can be attached with the date and time and time with the following template.

       Original name

To replace original file name.(Ex. DSC_0001)

       Date

To replace shoot date.(Ex. 2018-01-01)

       Time

To replace shoot time.(Ex. 13:01:10)

 

        Sequence number

Add the sequence number to the file name when archiving.

       Do not used

Do not append sequence number

       Append 3 digits sequence number

Append 001-999

       Append 3 digits sequence number

Append 0001-9999

       Append 3 digits sequence number

Append 00001-99999

       Append 3 digits sequence number

Append 000001-999999

 

        File rename template

Enter the model of the file to be renamed. Default value is DSC_%DATE%_%TIME%

The last line shows the directory and filename actually saved.

 

        Store the file in the directory

You can create directories when archiving. When checked, "Directory name template" is displayed.

The directory name can be attached with the years, month and day with the following template.

       Year

To replace Year.(Ex. 2018)

       Month

To replace Month(Ex. 01)

       Day

To replace Day(Ex. 29)

 

        Directory template

Enter the model of the directory to be created. Default value is %YEAR%.
